# TrendScope AI

AI-Powered Product Trend Analysis and Sales Forecasting Dashboard

## Overview

TrendScope AI helps businesses identify high-performing products,
forecast future revenue, and generate AI-based product recommendations
using historical sales data.

## Technologies Used

- Python
- Pandas
- SQLite
- SQL
- Power BI
- Excel
- Forecasting Models

## Project Architecture

Raw Data
   ↓
Data Cleaning
   ↓
Data Transformation
   ↓
SQLite Database
   ↓
SQL Analysis
   ↓
Power BI Dashboard
   ↓
AI Recommendation Engine

## Features

- Revenue Analysis
- Profit Analysis
- Product Intelligence
- Sales Forecasting
- AI Product Recommendations

## Dashboard Pages

1. Executive Summary
2. Revenue Analysis
3. Profit Analysis
4. Product Intelligence
5. Forecasting

## AI Recommendation Logic

AI Score =
(0.7 × Sales) + (0.3 × Profit)

Recommendation Levels:

- Strong Buy
- Buy
- Watch
- Avoid

## Business Insights

- Technology generates the highest revenue.
- West region contributes the highest profit.
- Canon imageCLASS products dominate sales.
- Forecast indicates future revenue growth.
